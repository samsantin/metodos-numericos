# Métodos Numéricos — Implementaciones en Python

Este repositorio contiene una recopilación de implementaciones en **Python** de los principales **métodos numéricos** vistos en un la clase de Álgebra Lineal Numérica en ITESO.
Cada método se encuentra dentro de un archivo Jupyter Notebook (`.ipynb`).

---

## Contenido

Los notebooks incluidos son:

### Métodos de Álgebra Lineal

* `gauss_jordan.ipynb` — Método de Gauss-Jordan
* `gauss_seidel.ipynb` — Método de Gauss-Seidel
* `metodo_jacobi.ipynb` — Método de Jacobi
* `factorizacion_lu.ipynb` — Factorización LU
* `factorizacion_chelseky.ipynb` — Factorización de Cholesky
* `gradiente_conjugado.ipynb` — Gradiente conjugado
* `algoritmo_qr.ipynb` — Método QR para autovalores

### Métodos para Solución de Ecuaciones

* `metodo_punto_fijo.ipynb` — Método de Punto Fijo
* `biseccion_puntofijo_newtonrhapson.ipynb` — Bisección, Newton–Raphson combinados
* `metodo_newton.ipynb` — Método de Newton-Raphson (forma directa)

### Métodos Iterativos para Autovalores

* `metodo_potencia.ipynb` — Método de la Potencia
* `potencia_inversa.ipynb` — Método de la Potencia Inversa

### Optimización Numérica

* `descenso_mas_rapido.ipynb` — Método de descenso más rápido
* `metodo_sobrerelajacion.ipynb` — Método SOR
